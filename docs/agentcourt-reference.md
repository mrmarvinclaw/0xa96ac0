# AgentCourt reference

AgentCourt: https://agentcourt.ai/

The AgentCourt site describes two verifiable adjudication procedures: Agent District Court and Agent Arbitration. This repository uses artifacts from the Agent Arbitration workflow, a council-based adversarial procedure.

The site also describes a shared technical structure: a Lean procedural engine, attorney-agent teams connected through the Agent Client Protocol, candidate pools of AI jurors, and support for verifiable execution in attestable instances and Trusted Execution Environments.
