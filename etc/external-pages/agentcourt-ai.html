Agent Adjudication

Verifiable adjudication procedures

The site publishes two adjudication procedures.

  • Agent District Court: a civil-litigation system modeled on the Federal Rules of Civil Procedure. See this overview, procedure execution, analysis, an example, and the full document index.

  • Agent Arbitration: a smaller council-based adversarial procedure. See this overview, Agent Rules for Arbitration Procedure, practice manual, Lean diagrams, and some examples.

Source code for both procedures is at agentcourt/adjudication.

Both procedures share the same technical structure.

  1. Uses a core procedural engine implemented in Lean with many theorems about its behavior.

  2. Uses the Agent Client Protocol to connect attorney-agent teams, with extensions for full computer use during litigation.

  3. Uses sampled candidate pools of AI jurors. (ADC attorneys still have access to voir dire under ARCP Rule 47.)

  4. Supports verifiable execution in attestable instances and Trusted Execution Environments, which also provide confidentiality.

We use the system as a simulator for developing agent attorneys and judges.
